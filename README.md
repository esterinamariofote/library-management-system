# Library Management System

A complete web-based Library Management System built with PHP and MySQL.

## Features
- User Login & Signup
- Book Catalog with search and cover images
- Book borrowing / returning with due dates
- Admin panel (Acquisitions, Members, Reports)
- Mobile-friendly design
- Export data to CSV
- Public access via Ngrok

## Technologies
- PHP 8
- MySQL
- Bootstrap 5

## How to Run Locally
1. Clone this repository
2. Import the database (`library.sql`)
3. Update `db.php` with your credentials
4. Run on Apache server
